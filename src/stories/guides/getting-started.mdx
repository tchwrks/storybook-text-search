# Getting Started Guide

Welcome to our design system! This guide will help you get up and running quickly.

## Installation

```bash
npm install @company/design-system
# or
yarn add @company/design-system
```

## Basic Setup

1. Import the necessary styles
2. Configure your theme
3. Wrap your application

## First Component

Here's a simple example using our Button component:

```jsx
import { Button } from '@company/design-system';

function App() {
  return <Button variant="primary">Click me!</Button>;
}
```
